# PYTHON SERVER

Server for streaming application

## Create Python Virtual Environment

### Windows

```powershell
py -m venv env
```

### Linux

```sh
python3 -m venv env
```

## Activate Python Virtual Environment and run server

### Windows

```sh
source env/Scripts/activate
py server.py
```

### Linux

```sh
source env/bin/activate
python3 server.py
```
